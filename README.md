# 스마트스토어 유입 대시보드

Google Sheets와 자동 연동되는 인터랙티브 대시보드. Vercel에 배포하면 URL 하나로 누구나 접속 가능합니다.

---

## 🚀 배포 가이드 (10분 소요, 1회만)

### 사전 준비

1. **GitHub 계정** (https://github.com) — 무료, 가입 5분
2. **Vercel 계정** (https://vercel.com) — GitHub로 로그인 가능, 무료
3. **Google Sheets** — 데이터 누적용

---

### Step 1. Google Sheets 세팅

1. 새 스프레드시트 생성
2. **1행에 컬럼 헤더 입력** — 네이버 raw 다운로드 형식 그대로:
   ```
   날짜 | 채널속성 | 채널그룹 | 채널명 | 채널상세 | 고객수 | 유입수 | 광고비 | 페이지수 | 유입당 페이지수 | 결제수(마지막클릭) | 유입당 결제율(마지막클릭) | 결제금액(마지막클릭) | 유입당 결제금액(마지막클릭) | ROAS(마지막클릭) | 결제수(+14일기여도추정) | 유입당 결제율(+14일기여도추정) | 결제금액(+14일기여도추정) | 유입당 결제금액(+14일기여도추정) | ROAS(+14일기여도추정)
   ```
   ※ 대시보드는 **9개 컬럼만 사용**합니다 (날짜, 채널속성, 채널그룹, 채널명, 채널상세, 유입수, 광고비, 결제수(마지막클릭), 결제금액(마지막클릭)). 나머지는 있어도 무시되니 raw 그대로 붙여넣으면 됩니다.

3. 매일 네이버에서 raw 다운 → 시트 맨 아래에 **값으로 붙여넣기** (Ctrl+Shift+V)
4. **공유 설정**: 우측 상단 [공유] → "일반 액세스"를 **'링크가 있는 모든 사용자'**(뷰어)로 변경
5. **주소창의 시트 URL을 복사해둡니다** (예: `https://docs.google.com/spreadsheets/d/abc123.../edit#gid=0`)

---

### Step 2. GitHub에 코드 업로드

1. https://github.com/new 에서 새 저장소 생성 (이름 자유, 예: `smartstore-dashboard`)
2. "Add file" → "Upload files" → 압축 푼 폴더의 **모든 파일/폴더 드래그**
3. 하단 [Commit changes] 클릭

---

### Step 3. Vercel 배포

1. https://vercel.com/new 접속 → GitHub 로그인
2. 방금 만든 저장소 옆 [Import] 클릭
3. "Configure Project" 화면에서:
   - **Framework Preset**: Next.js (자동 인식됨)
   - **Environment Variables** 펼치고 추가:
     - **Name**: `SHEET_URL`
     - **Value**: Step 1에서 복사한 시트 URL 붙여넣기
4. [Deploy] 클릭 → 1~2분 대기
5. 완료되면 **`https://your-project.vercel.app`** 같은 URL이 생깁니다 → 누구든 이 URL로 접속

---

## 📅 매일 운영

1. 네이버 스마트스토어에서 raw 데이터 다운
2. Google Sheets 맨 아래에 데이터 추가
3. 끝 — 대시보드 접속자가 새로고침하면 최신 데이터 반영됨

데이터 추가 후 즉시 반영됩니다 (대시보드 접속 시 매번 시트에서 가져옴).

---

## 🔒 보안 메모

- 시트 URL은 Vercel 환경변수로 관리되어 **클라이언트 코드/네트워크에 노출되지 않습니다**
- 시트 자체는 "링크가 있는 모든 사용자" 뷰어 권한이 필요하지만, URL을 모르면 접근 불가
- 대시보드 URL 자체는 누구나 접속 가능 — 내부용으로만 공유하세요
- 외부에 공개되면 안 되는 경우, Vercel Pro의 비밀번호 보호 또는 [vercel-protect](https://vercel.com/docs/security/deployment-protection) 설정 가능

## 🛠 로컬 개발 (선택)

```bash
npm install
echo "SHEET_URL=https://docs.google.com/spreadsheets/d/..." > .env.local
npm run dev
```

http://localhost:3000 접속.

## 📊 대시보드 기능

- **필터링**: 채널속성/유입유형(광고·오가닉)/채널그룹/채널명/채널상세 — 체크박스로 집계 기준 선택, 항목 클릭으로 필터링
- **조회 기간**: 시작일~종료일 date picker
- **집계 단위**: 자동(기간 길이로 결정) / 일간 / 주간 / 월간 토글
- **KPI**: 유입수, 광고비, 결제수, 결제금액 (광고/오가닉 분리 표시)
- **광고 vs 오가닉 비교**: 각 KPI의 광고/오가닉 비중 + 기간 내 성장률
- **트렌드 차트**: 유입 추이(스택 막대) + 매출 라인, 매출/결제수 분리 라인
- **채널별 상세 테이블**: 선택한 차원으로 집계, 매출 큰 순 정렬
